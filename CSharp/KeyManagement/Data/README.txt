erstellen Sie selber unter diesem Projekt eine Datenbank.
1. Ordner "Data" rechts anklicken -> Add -> new Item
2. unter 'visual c# items' -> data -> service based database (mit Endung .mdf) und benennen Sie die DB.
3. doppel klick auf der gerade generierte DB Datei und dann sehen Sie das 'Server Explorer' Panel
4. auf '<DBName>.mdf' in diesem Panel rechts klicken -> modify connection
5. auf pop-up Panel, unter Advanced, kopieren Sie die untenstehende Connectionstring
